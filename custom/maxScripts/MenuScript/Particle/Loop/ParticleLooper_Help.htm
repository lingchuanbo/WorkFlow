Particle Looper 1.0
Written By: Jason Hayes
Date: 02/12/2002
Developed for 3D Studio MAX 4.2


Particle Looper is a script that will loop any of the following supported particle system types:
  • Blizzard
  • PArray
  • PCloud
  • Super Spray

The advantage of using this over Peter Watje's "Particles +" plugin is that you can use some of the default MAX particle systems in case you need to use instanced geometry. What sucks is that it's a big cheese workaround, but it does the trick! :)

Description

Pick Base Particle System - Use this to choose the base system.
Emitter Start / Stop Frame Range - The number of frames between the base Emitters Start / Stop frame. It should automatically get this number. This number MUST be the EXACT same amount of frames as the length of your animation.
Consistent wire color - If this option is checked, the script will make all systems the same wire color as the base.
Loop Particle System - Generates the loop.
Delete Looped System - The Particle Looper script generates copies of the base system and adds a prefix of "PL_". Clicking this will remove any objects with that prefix.

 

How in the heck do I use this thing?

  1. The Emitter Start and Stop range MUST be the EXACT same number of frames as the length of your animation. Example: animation length = 900f, Emitter Start = -300, Emitter Stop = 600. The range between the start and stop is exactly 900 frames.
  2. I've found if you start the Emitter at a negative frame such as -300, it works much better than starting at a positive frame.
  3. For a 900 frame animation, I've found that the Display Until and Life values should be set to 2000. This number will probably go up or down expodentially depending on the length of your animation.
